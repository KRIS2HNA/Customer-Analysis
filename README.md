# Customer Analysis Project

## Overview
This project performs comprehensive customer analysis using Jupyter notebooks and CSV data files to extract insights and patterns from customer data.

## Project Structure
- **customer analysis.ipynb** - Main analysis notebook with data exploration and visualization
- **headbrain.csv** - Customer dataset used for analysis
- **Untitled.ipynb** - Additional analysis notebook

## Getting Started

### Requirements
- Python 3.x
- Jupyter Notebook
- pandas
- numpy
- matplotlib/seaborn (for visualization)

### Installation
1. Clone or download this repository
2. Install required packages: `pip install jupyter pandas numpy matplotlib seaborn`
3. Open Jupyter Notebook: `jupyter notebook`

## Usage
Run the notebooks in the following order:
1. `customer analysis.ipynb` - Main analysis
2. `Untitled.ipynb` - Supporting analysis (if needed)

## Data
- **headbrain.csv** - Contains customer/product data for analysis

## Analysis Includes
- Customer segmentation
- Data exploration and visualization
- Statistical analysis
- Pattern identification

## Notes
All analysis and visualizations are contained within the Jupyter notebooks.
